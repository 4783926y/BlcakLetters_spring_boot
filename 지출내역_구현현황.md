# 📊 프로젝트 기능 구현 현황 분석 보고서 (Feature Status)

이 문서는 현재 Spring Boot 백엔드 애플리케이션(`BlackLetters_spring_boot`)에 구현되어 있는 기능과 구현되어 있지 않은 기능을 분류하고, 특히 영수증 지출 내역의 조회, 수정, 삭제 기능을 포함하여 상세히 분석한 문서입니다.

---

## 1. 종합 요약 (Summary)

전체 **18개 핵심 기능** 중 구현 상태는 다음과 같습니다.

* **구현 완료 (7개)**: 즉시 사용 가능한 API와 비즈니스 로직이 완성된 상태입니다.
* **일부 구현 (2개)**: 기능이 존재하나 세부 흐름(승인 단계, 단일 상세 조회 등)에서 보완이 필요한 상태입니다.
* **미구현 (9개)**: 추가 개발이 필요한 상태입니다.

| 구분 | 개수 | 해당하는 기능 리스트 |
| :--- | :---: | :--- |
| **구현 완료** | **7** | 카카오 연동 로그인, 카테고리 목록 조회, 커스텀 카테고리 추가, 영수증 이미지 업로드, 영수증 OCR 분석 및 상태 관리, 영수증 정보 자동 추출 (기본 및 상세), 월별 카테고리 예산 설정 |

## 3. 영수증 지출 내역 (조회, 수정, 삭제) 상세 분석 및 보완 제안

### [1] 지출 내역 조회 (일부 구현)
* **현재 상태**: `GET /api/v1/receipts`를 호출하여 사용자의 전체 영수증 리스트를 조회할 수 있습니다.
* **보완 제안**: 영수증 상세 화면으로 진입할 때 개별 영수증 ID로 영수증 정보와 구매한 세부 품목(`ReceiptItem` 목록)을 함께 조회하는 **단일 상세 조회 API**가 필요합니다.
  ```text
  GET /api/v1/receipts/{receiptId}
  ```

### [2] 지출 내역 수정 (미구현)
* **현재 상태**: 현재는 OCR로 자동 추출되어 데이터베이스에 저장된 값을 수정할 방법이 전혀 없습니다. (가맹점명 오타, 금액 오인식 등 발생 시 대처 불가)
* **보완 제안**: 영수증 내용 및 세부 품목 정보를 수정하여 다시 저장할 수 있는 **수정 API**를 추가해야 합니다.
  ```text
  PUT /api/v1/receipts/{receiptId}
  ```

### [3] 지출 내역 삭제 (미구현)
* **현재 상태**: 등록된 지출 내역이나 영수증을 삭제할 수 있는 기능이 정의되어 있지 않습니다.
* **보완 제안**: 영수증 데이터를 삭제할 때, 해당 영수증에 종속된 세부 품목(`ReceiptItem`)도 함께 지워지도록 Cascade 삭제 또는 영속성 전이(JPA `orphanRemoval = true`)를 적용한 **삭제 API**를 구현해야 합니다.
  ```text
  DELETE /api/v1/receipts/{receiptId}
  ```
